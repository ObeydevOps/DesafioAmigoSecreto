# DesafioAmigoSecreto
Bem-vindo ao repositório do Amigo Secreto! Este projeto contém um script para organizar um sorteio de amigo secreto, garantindo que ninguém tire a si mesmo e que o sorteio seja justo e aleatório.

O script fornecido realiza o sorteio de um grupo de participantes para um jogo de amigo secreto. Ele embaralha os nomes dos participantes e associa cada pessoa a outra, garantindo que:
Cada participante receba um amigo secreto diferente de si mesmo.
O sorteio seja completamente aleatório.
O resultado do sorteio é exibido no console, mostrando quem tirou quem.
